Machine Learning

Cotton Leaf Disease Detection

Our project uses technology to help cotton farmers. We use a type of computer program called CNNs to find and identify diseases in cotton plants. We found that the ResNet-152v2 model works best for this task.

We improved this model to make it faster and more accurate. It helps farmers know if their plants are sick, so they can take action early.

We made a simple web app using Flask where users can upload pictures to get disease information. This helps farmers make better decisions about their crops.
